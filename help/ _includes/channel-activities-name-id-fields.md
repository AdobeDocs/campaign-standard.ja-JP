# チャネルアクティビティ名と ID フィールドに関する注意事項 {#note-name-id-fields}

>[!NOTE]
>
>アクティビティ名とアクティビティ ID は両方ともインターフェイスに表示されますが、メッセージ受信者には表示されません。
>
>例えば、Adobe Analyticsとの統合時など、不一致が生じるのを避けるために、ID フィールドに空白を含めないようにしてください。

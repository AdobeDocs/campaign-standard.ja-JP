# SMS レポート{#sms-report}

**SMS**&#x200B;レポートは、配信済みや直帰率など、SMS配信の詳細を提供します。

![](assets/dynamic_report_sms.png)

**SMSサマリ**&#x200B;の表、グラフ、およびサマリ番号には、送信されたSMS配信で使用可能なデータが含まれます。

* **処理済み/送信済み**:送信されたSMSの数。
* **配信済み**:配信されたSMSの数。
* **バウンス+エラー**:配信できなかったメッセージの数です。

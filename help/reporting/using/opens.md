# 開封数{#opens}

**[!UICONTROL Opens]** レポートは、受信者が最も閲覧した配信を特定します。

![](assets/delivery_reports_opens.png)

次の条件に基づいて、4 つの表とグラフによって、メールを開封した受信者の合計数を分類します。

* ブラウザー
* デバイス
* プラットフォーム
* ドメイン

**[!UICONTROL TOP 5]**&#x200B;のテーブルとグラフには、配信されたメッセージの数が最も多い配信が表示されます。

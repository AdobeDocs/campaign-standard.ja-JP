# 指標の計算{#indicator-calculation}

>[!NOTE]
>
>大量の分析とリアルタイム分析をより適切に処理および管理するために、動的レポートでは、明確な数の予測に対して概算の集計を使用します。 近似集計オファーのメモリ使用量に制限があり、多くの場合、正確な計算よりも高速です。

次の表に、様々なレポートで使用されるインジケータのリストと、配信のタイプに応じたその計算式を示します。

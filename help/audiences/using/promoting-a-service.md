# サービスのプロモーション{#promoting-a-service}

サービスにサブスクリプションを提供するには、いくつかの方法があり、訪問者にサブスクリプションを管理する機能があります。

Campaignを使用して、次の方法でサービスを宣伝できます。

* [ サービスの購読または購読解除リンクをメールに挿入](../../designing/using/links.md#inserting-a-link)。

* [購読または購読解除ランディングページへのリンクをメールに挿入](../../designing/using/links.md)。 この場合、サービスは、関連するランディングページのプロパティで直接参照する必要があります（[ ランディングページをサービスにリンクする](../../channels/using/configuring-landing-page.md#linking-a-landing-page-to-a-service)を参照）。

  >[!NOTE]
  >
  >購読者に購読解除の可能性を与えることも重要です。 これを行うには、新しい購読者に自動的に送信される確認メール（サービスプロパティで定義）に、今後のニュースレターメールに含まれるサービス <b>購読解除リンク </b>を挿入します。

* web サイトでサブスクリプションまたは購読解除ランディングページを利用できるようにします。 ランディングページにアクセスできるURLには、関連するサービスなどのパラメーターと、それにアクセスするプロファイル IDを指定する必要があります。 このIDは、ランディングページのパラメーターで定義できます（[ ランディングページの設定](../../channels/using/configuring-landing-page.md)を参照）。

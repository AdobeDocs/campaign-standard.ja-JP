# データモデルの紹介{#data-model-introduction}

Adobe Campaignには、事前定義されたデータモデルが付属しています。

このデータモデルは、既存のリソースに [新しいリソース](../../administration/using/users-management.md#functional-administrators) や拡張機能を追加できる管理者が変更できます。

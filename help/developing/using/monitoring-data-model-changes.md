# データモデルの変更の監視{#monitoring-data-model-changes}

この **[!UICONTROL Diagnosis]** メニューを使用すると、アプリケーションで生成される様々な技術オブジェクトを調べて、それらを分析できます。

>[!NOTE]
>
>このメニューの画面は読み取り専用です。

![](assets/diagnostic.png)

次の種類のオブジェクトを表示できます。

* データスキーマ
* Web ページ
* フィルター
* ナビゲーション
* バッチジョブ

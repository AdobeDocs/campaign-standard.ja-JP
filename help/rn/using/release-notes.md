## リリース 26.1 （LA） {#26.1}

>[!AVAILABILITY]
>
>このリリースは、一連の組織のみが使用できます（限定提供）。詳しくは、アドビ担当者にお問い合わせください。

### セキュリティ関連の修正 {#26-1-security}

* このリリースでは、セキュリティ上の問題が修正されています。
* このリリースには、Debian 12 および PostgreSQL 17 のセキュリティアップグレードが付属しています。

### その他の修正点 {#26-1-fixes}

* 動的レポートを改善し、堅牢性を向上し、データの不一致を削減しました。（CAMP-58964）
* IMS ログインメカニズムが改善され、信頼性が向上しました。 （CAMP-59418）
* ページ分割されたリクエストの URL 処理が改善され、AEM コンテンツの統合が改善されました。 （CAMP-59540）
* 自動ハッシュ付きのオプション画面でパスワードタイプの入力がサポートされるようになりました。 （CAMP-60206）
* 一時ワークフローで間違ったシーケンス ID が使用されていた問題を修正しました。 （CAMP-59824）
* AEM Campaign コンテンツの取得時にエラーが発生する可能性がある問題を修正しました。 （CAMP-59926）

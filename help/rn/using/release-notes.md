## リリース 26.2 {#26.2}

### セキュリティ関連の修正 {#26-2-security}

* このリリースでは、セキュリティ上の問題が修正されています。
* このリリースには、Debian 13 および PostgreSQL 17 のセキュリティアップグレードが含まれています。

### その他の修正点 {#26-2-fixes}

* 動的レポートを改善し、堅牢性を向上し、データの不一致を削減しました。 （CAMP-58964）
* IMS ログインメカニズムを改善して信頼性を向上しました。 （CAMP-59418）
* ページ分割されたリクエストの URL 処理を改善し、AEM コンテンツとの統合を強化しました。 （CAMP-59540）
* オプション画面にパスワード入力タイプのサポートを追加し、自動ハッシュ化に対応しました。 （CAMP-60206）
* 一時的なワークフローで正しくないシーケンス ID が使用されていた問題を修正しました。 （CAMP-59824）
* AEM キャンペーンコンテンツを取得する際にエラーが発生する可能性がある問題を修正しました。 （CAMP-59926）

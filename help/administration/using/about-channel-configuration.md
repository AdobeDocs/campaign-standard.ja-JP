# チャネル設定について{#about-channel-configuration}

Adobe Campaignのすべてのコミュニケーションチャネルは、メッセージを効果的に送信できるように正しく設定されている必要があります。

様々な設定は次のとおりです。

* ルーティング、処理ルール、配信品質パラメーターなど、チャネルの技術的設定。
* メッセージのテンプレートとプロパティ： すべてのメッセージはテンプレートに基づいています。 Adobe Campaignには、アクティブ化されたすべてのチャネルに対する一連の定義済みテンプレートが用意されています。 これらのテンプレートは複製してカスタマイズできます。
* メッセージのプロパティ： 新しいメッセージを作成する際に、ニーズに応じてプロパティを微調整できます。
* 一部のチャネルでは、「電子メールのBCC」など、オプションの追加設定を提供しています。

**[!UICONTROL Channel]** メニューには、様々なアプリケーションチャネル（電子メール、SMS）にリンクされている技術的なパラメーターと、タイポロジおよび強制隔離の管理が再グループ化されます。

管理者は、**[!UICONTROL Administration]** > **[!UICONTROL Channels]** メニューでこれらの設定にアクセスできます。
